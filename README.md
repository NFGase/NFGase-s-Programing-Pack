### 下载 & 使用教程

**下载**：拉动到该项目主页的上方，点击右上角的按键“Code”，随后在出现后的下拉界面中点击“Download ZIP”，下载完毕后打开所下载的位置，右键解压即可。

**使用**：打开解压后的文件夹，选择并打开你需要用的资料^1即可。

^1: 有些 $LaTeX$ 语法GitHub不支持，可以自己下载一个MarkDown编辑器来看，这里推荐MarkText，就是下载慢。
